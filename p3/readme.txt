Compile diskinfo.c using the command "make compile1" and run using the command "make a ARG1=disk2.IMA"
Compile disklist.c using the command "make compile2" and run using the command "make b ARG1=disk2.IMA"
Compile diskget.c using the command "make compile3" and run using the command "make c ARG1=disk2.IMA ARG2=ANS1.PDF"
Compile diskput.c using the command "make compile4" and run using the command "make d ARG1=disk2.IMA ARG2=FOO.TXT"
